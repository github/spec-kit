# v0.12.5+teamwork.1

`v0.12.5+teamwork.1` is the first pinned AI Team SDD distribution based on
upstream Spec Kit `v0.12.5` (`12efa87`).

## Distribution Scope

- AI Team plain-language Intake and deterministic work-item routing;
- separate bug-fix and feature/new-project workflows;
- Standard and Compact planning with explicit human Gates;
- role-isolated SDD handoffs;
- durable Work Context Packages and resume guidance;
- permission envelopes and source/Code Graph impact controls;
- portable checks and Evidence Board output;
- local, department, and enterprise memory consolidation contracts;
- release-scoped archive and reusable lesson promotion;
- Codex, Claude Code, Cursor Agent, and Trae integration paths.

## Version Policy

- Git tag: `v0.12.5+teamwork.1`
- Python package version: `0.12.5+teamwork.1`
- Upstream base: `github/spec-kit@v0.12.5`
- Automatic `specify self check/upgrade`: not advertised for this pinned release

Install or repair the fixed release with:

```bash
uv tool install specify-cli --force \
  --from git+https://github.com/EuphoriaYan/spec-kit.git@v0.12.5+teamwork.1
```

Project integrations, extensions, presets, and workflows remain explicit
installation choices after the CLI is installed.
